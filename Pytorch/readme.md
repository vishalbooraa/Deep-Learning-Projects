## 🚀 Repository Contents

Currently, this repository features a comprehensive Jupyter Notebook (`pytorch_tutorial.ipynb`) covering the following core areas:

* **Tensor Operations:** Creating, reshaping, and managing tensors, including device placement for GPU (CUDA) acceleration.
* **Image Tensor Formatting:** Understanding and manipulating the standard `[B, C, H, W]` (Batch, Channel, Height, Width) image tensor format for computer vision tasks.
* **Autograd & Computational Graphs:** Leveraging PyTorch's automatic differentiation (`.backward()`, `.grad`) and managing gradient tracking (`torch.no_grad()`).
* **Custom Neural Networks:** Building a basic Convolutional Neural Network (CNN) architecture using `torch.nn.Module` and `nn.Sequential` with `Conv2d` and `Linear` layers.
* **Optimization:** Initializing and applying the Adam optimizer (`torch.optim.Adam`) to update model weights.
